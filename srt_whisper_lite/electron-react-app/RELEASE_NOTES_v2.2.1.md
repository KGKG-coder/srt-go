## 🚀 使用方式

### 啟動應用
```bash
# 直接執行
"SRT GO - AI Subtitle Generator.exe"

# 或從開發環境
npm start
```

### 處理音頻檔案
1. 拖放音頻/影片檔案到界面
2. 選擇 AI 模型 (Large/Medium/Base)
3. 設定語言 (自動檢測/指定語言)
4. 點擊開始處理
5. 生成 SRT 字幕檔案
